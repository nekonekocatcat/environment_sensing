# 環境コレクション

小型軽量な環境センサを持ち歩いて自分の生活する環境をコレクションするゲームです
取得した環境データをノーマル環境・レア環境の2種類に分類しています

| No. | ノーマル環境 | レア環境 |

| 2 | リビングまったり環境 | クラブわいわいレア環境 |
| 3 | 夜ふかしの薄暗い部屋 | 工事現場みたいなレア環境 |
| 4 | 早朝の静けさ環境 | 真夏の密室レア環境 |
| 5 | 空気こもり気味環境 | 南国リゾートレア環境 |
